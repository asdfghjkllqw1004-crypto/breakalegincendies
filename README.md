# INCENDIES Profile Archive

GitHub/Vercel 업로드용 최종 파일입니다.

## GitHub에 올리는 방법
1. 이 압축파일을 풀어주세요.
2. GitHub 저장소 `performance-profile-site`에 들어갑니다.
3. `Add file` → `Upload files`를 누릅니다.
4. 압축을 푼 폴더 안의 파일/폴더 전체를 업로드합니다.
   - `index.html`
   - `package.json`
   - `vite.config.js`
   - `src` 폴더
   - `public` 폴더
5. Commit changes를 누릅니다.

## Vercel 설정
- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
